# 🛡️ SkillGate: Secure Your Agent's Soul

**SkillGate** is the industry-standard security layer for the agentic era. Before you let an AI Agent touch your files or execute code, make sure it passes through the Gate.

## Why SkillGate?
As the OpenClaw ecosystem explodes, malicious skills are on the rise. SkillGate provides a **Three-Layer Defense** to ensure your automation scales safely.

### 🚀 Core Features
- **L1 Static Analysis**: Instant detection of hardcoded keys and dangerous commands.
- **L2 Dynamic Sandbox**: (Pro) Safely run skills in an isolated environment to monitor real-time behavior.
- **Auto-Hardening**: One-click system security reinforcement based on OWASP ASI standards.

## Usage

### 🔍 Vet a New Skill
```bash
# Analyze a skill before installation
call skillgate.vet(target="github.com/user/new-skill")
```

### 🛡️ Harden Your System
```bash
# Apply immediate security patches
call skillgate.harden()
```
