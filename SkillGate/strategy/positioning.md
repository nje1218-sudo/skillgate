# SkillGate Positioning

## One-liner
SkillGate 是 OpenClaw / agent 生態的 Security + Ops + Governance layer。

## What it is not
- 不是單純 restart 腳本
- 不是只有 healthcheck UI
- 不是只有 skill 市集

## What it is
- 入庫門禁
- 可靠性層
- incident reporting / audit layer
- 升版 diff gate

## Core wedge
先用電子書 + 模板包切入，再往 Vetting / Guardian / Reports 擴張。
