# SkillGate Roadmap

更新時間：2026-03-15

## V1：MVP
目標：先證明 skill 上線前的安全門禁能成立。

### 功能
- Skill Intake
- L1 Static Vetting
- L2 Sandbox Runtime Check
- Policy Gate
- Trusted Registry
- Least-Privilege Execution Constraints
- Audit Report

### 商業價值
- 能 demo
- 能賣概念驗證
- 能開始找 design partner

---

## V1.5：治理強化
目標：把單次審查流程，補成可持續治理能力。

### 功能
- Diff Gate
- Dependency / License Tracking
- Explainable Findings
- Restricted Admission Profiles
- 基礎 Drift Detection

### 商業價值
- 更像團隊流程工具
- 更適合企業試點
- 更能支持 repeat review

---

## V2：企業平台化
目標：把 SkillGate 從門禁工具升級成供應鏈與執行治理平台。

### 功能
- Source Reputation Scoring
- Publisher Trust / Identity Verification
- Signature / Provenance / Immutable Version Pinning
- SkillGraph
- Learned Behavior Baseline
- Auto Downgrade / Quarantine / Rollback
- Incident Workflow / Ticketing Integration

### 商業價值
- 更高單價
- 更強 enterprise security 敘事
- 更清楚的 platform moat

---

## 產品節奏原則
1. 先證明能審、能限權、能留痕
2. 再補 repeatable governance
3. 最後補 enterprise-grade automation 與 trust graph

## 一句總結
**先活下來，再變性感。先把 MVP 做出來，再談企業級全生命週期治理。**
